# Portfolio Professionnel - Marina

## Description
Ce portfolio professionnel moderne présente les compétences, projets et expériences de Marina, développeuse web passionnée.

## Fonctionnalités
- Design moderne et responsive
- Navigation fluide avec menu burger mobile
- Sections complètes : À propos, Services, Projets, Formation, Contact
- Animations et effets visuels
- Formulaire de contact fonctionnel
- Téléchargement de CV
- Optimisé pour tous les appareils

## Technologies utilisées
- HTML5 sémantique
- CSS3 avec Flexbox et Grid
- JavaScript vanilla
- Font Awesome pour les icônes
- Google Fonts (Inter)

## Structure des fichiers
```
portfolioprofessionel/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
├── image/
│   └── marina.png
├── cv/
│   └── (fichier CV à ajouter)
└── README.md
```

## Installation
1. Cloner ou télécharger le projet
2. Ajouter votre fichier CV (PDF) dans le dossier `cv/`
3. Personnaliser les informations dans `index.html`
4. Ouvrir `index.html` dans un navigateur

## Personnalisation
- Modifier les informations personnelles dans `index.html`
- Ajouter vos vrais projets et images
- Personnaliser les couleurs dans `css/style.css`
- Ajouter votre CV dans le dossier `cv/`

## Responsive Design
Le site est entièrement responsive et s'adapte à tous les types d'écrans :
- Desktop (1200px+)
- Tablette (768px - 1199px)
- Mobile (moins de 768px)
